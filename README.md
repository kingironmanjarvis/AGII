# AGII V10 — Advanced AI Agent

The most advanced autonomous AI agent, built with Groq's LLaMA 3.3 70B.

## Features
- 🧠 Persistent memory across sessions
- 🌐 Real-time web search
- ⚡ Code execution engine
- 🔧 Skills system (create & reuse functions)
- 🗺 Strategic planning & multi-step reasoning
- 📄 File read/write & URL fetching
- 🤖 15+ tools, autonomous chaining up to 8 iterations
- 5 AI models (LLaMA 3.3 70B, Mixtral, Gemma 2, LLaMA 4 Scout)
- SSE streaming responses
- Beautiful dark UI with thinking transparency

## Deploy
See render.yaml — deploy both backend (Node.js) and frontend (static) to Render.

## Local Dev
```bash
cd backend && npm install && GROQ_API_KEY=xxx node server.js
```
